\chapter{GIT HISTORY }
To ensure proper version management and easy distribution, all game builds have been uploaded to Git. This allows for efficient tracking of updates, bug fixes, and improvements over time. By using Git, different versions of the game can be accessed, making it easier to roll back changes if needed. The repository serves as a centralized location for storing and sharing the game builds, ensuring that the latest version is always available for testing and distribution.
