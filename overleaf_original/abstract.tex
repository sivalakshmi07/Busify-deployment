 Debug Defuse is an interactive coding-based maze game developed using the Unity game engine. The game is designed to enhance players' programming and problem-solving abilities by integrating real-time coding challenges within an engaging gameplay experience. In the game, the player navigates through a 3D maze environment, where multiple checkpoints are strategically placed. As the player reaches a checkpoint, a coding question appears on the screen through the Unity UI system. The player must write the solution to the coding problem in a text input field provided within the game's interface.
\\\\
Once the player submits their code, it is sent to the JDoodle API, which acts as an online compiler and code execution engine. The JDoodle API compiles the submitted code, executes it, and checks the output against predefined test cases. If the solution passes all the test cases, the checkpoint is cleared, and the player can continue progressing through the maze. If the player fails to provide the correct output, they must try again until the correct solution is submitted. This mechanism ensures players not only understand coding concepts but also apply them in a time-sensitive, gamified scenario.\\\\
