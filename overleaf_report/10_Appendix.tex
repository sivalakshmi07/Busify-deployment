\section{Minimum Software Requirements}
\medskip
\textbf{Operating System}: Windows 10/11
\\\\
\textbf{File Extraction}: WinRAR or 7-Zip

\section{Minimum Hardware Requirements}
\textbf{Processor}:  Intel Core i5 (or AMD equivalent) or higher for efficient game execution and compilation of code via the JDoodle API.
\\\\
\textbf{RAM}: 8 GB (16 GB recommended for smoother performance)
\\\\
\textbf{Graphics Card}:A dedicated GPU such as NVIDIA GTX 1050 or higher is recommended for rendering 3D maze environments smoothly.
\\\\
\textbf{Storage}: 500 MB free space
\\\\
\textbf{Display}: A full HD (1920x1080) resolution screen for clear visuals and UI interaction.
\\\\
\textbf{Input Devices}: Keyboard and mouse for navigation, coding input, and game interaction.
\\\\
\textbf{Network Connectivity}: A stable internet connection to send and receive code execution results from the JDoodle API.
\\\\
